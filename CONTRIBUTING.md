# Contributing to Haiti City Portal

Thank you for your interest in contributing to the Haiti City Portal project! We welcome code contributions, bug reports, and feature requests.

## Contributor License Agreement (CLA)

By submitting a contribution to this project (including Pull Requests, Issues, and Code), you agree to the following terms:

**"By submitting a contribution to this project, you agree that Haiti City Portal Project has a perpetual, world-wide, non-exclusive license to use, modify, and sub-license your contribution commercially, with the intent of supporting project contributors."**

This ensures we can maintain the project sustainability while keeping the core open for non-commercial use. All code contributions are licensed under **BSL 1.1** (see LICENSE).

## How to Contribute

1. **Fork the repository** provided.
2. **Create a feature branch**.
3. **Submit a Pull Request** with a clear description of changes.
4. **Follow the code of conduct** and maintain professional interactions.
5. **Include copyright headers** in all new `.ts` and `.tsx` files.
6. **Add yourself** to the [Contributors](#contributors) list below.

## Why Vwa?

[Vwa](https://haitiansintech.com) is the professional verification platform for Haitian technologists. By linking your profile:
- You build a verified track record of civic tech contributions.
- You join the network of Haitian professionals building digital infrastructure.
- You help establish a new standard for transparency in Haitian leadership.

We encourage all contributors to add their Vwa profile to their GitHub bio.
